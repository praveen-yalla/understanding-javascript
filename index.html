Lexical Environment

Where something sits physically in the code you write and what's around it.

In other words, in programming languages, where you to write your code is very important.

Execution Context

A wrapper to help manage the code that is running.

There are lots of lexical environments. Which one is currently running is managed via execution contexts.The wrapper can contain things beyond what you've written in your code.
